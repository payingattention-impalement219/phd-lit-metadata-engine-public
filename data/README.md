# Data Directory

This directory is intentionally empty in the public repository.

The local app stores SQLite databases, manual citation imports, and temporary metadata files here. These files may contain private search logs, licensed database exports, reviewer notes, and local paths, so they are ignored by Git.

Use this folder for your local working data only. Do not commit real `.sqlite`, `.csv`, `.xlsx`, `.ris`, `.bib`, `.jsonl`, or other exported review data unless you have deliberately prepared an anonymized public dataset.

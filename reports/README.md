# Reports Directory

This directory is intentionally empty in the public repository.

The app and notebook write local outputs here, including CSV, Excel, JSONL, TXT, RIS/BibTeX-derived summaries, figures, screening workbooks, and protocol tables. These generated files may contain search results, reviewer decisions, notes, local paths, or database exports, so they are ignored by Git.

Public releases should include only this README and any intentionally anonymized example files. Keep real review outputs private unless they have been checked, anonymized, and approved for sharing.
